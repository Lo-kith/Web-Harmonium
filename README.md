Experience the **soul of a traditional harmonium** — now reimagined for the modern web.
Smooth, responsive, and crafted with ❤️ for music lovers and creators.

## 🎵 Features That Feel Alive

### 🎼 Musical Core

* 🎶 Studio-quality harmonium sound
* 🎹 Real-time key response (ultra low latency)
* 🎼 Sargam (Sa Re Ga Ma) + Western notes
* 🎧 Immersive reverb & sound layering

### 🎨 Experience & Design

* ✨ Smooth animations & modern UI
* 🌈 Gradient-rich, glassmorphism design
* 📱 Fully responsive (mobile → desktop)
* 🧠 Beginner-friendly + pro-ready

### 🎛️ Power Controls

* 🔊 Dynamic volume control
* 🎵 Smart transpose system
* 🎚️ Full octave range
* 🎹 MIDI keyboard support

## 🚀 Getting Started

```bash
git clone https://github.com/Lo-kith/web-harmonium.git
cd web-harmonium
npm install
npm run dev
```

👉 Open: `http://localhost:3000`
